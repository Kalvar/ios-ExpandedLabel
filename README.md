## Supports

ExpandedLabel can easy fit and auto expand height of label with the inputed text, it supports iOS 6 and iOS 7.

## How To Get Started

``` objective-c
- (void)viewDidLoad
{
    [super viewDidLoad];
    self.expandedLabel.text = @"Hello\nApps\nWorld\nLaLa\n";
    [self.expandedLabel autoExpandHeight];
}
```
